# Miri Look Recommendation Modes and Premium Add-ons

> 기준일: 2026-06-26
> 목적: 헤어 추천 기준을 2개 모드로 분리하고, 코디/메이크업 확장 상담을 프리미엄 상품으로 발전시키기 위한 실행 기준을 정리한다.

## 1. 추천 기준 분리

### 현재 기장 기준

- 목적: 미용실 현장 상담에서 바로 쓸 수 있는 스타일 추천
- 기준: 업로드 사진에 보이는 앞머리, 윗머리, 옆머리, 뒷머리 기장으로 가능한 컷, 펌, 다운펌, 컬러를 우선 추천
- 제외: 상당한 기장 성장 없이는 불가능한 스타일
- 예외: 고객이 직접 선택한 스타일이면 포함 가능하되, 추천 사유에 기장 성장 필요성을 명시
- 대표 사용처: 미용실 상담, 디자이너 제안, 당일 시술 가능 옵션 비교
- 화면 라벨: `현재 기장으로 가능한 스타일`
- 서버 값: `recommendationMode=current-length`

### 얼굴 적합 기준

- 목적: 개인이 장기적으로 어떤 스타일이 어울리는지 탐색
- 기준: 현재 기장보다 얼굴형, 두상, 분위기, 이목구비 밸런스, 원하는 인상을 우선
- 포함: 기장을 길러야 하는 스타일, 전환 컷, 펌, 컬러, 탈색이 필요한 스타일
- 필수: 실제 시술까지 필요한 과정과 주의점을 추천 사유에 명시
- 대표 사용처: 개인 탐색, 장기 스타일 방향, 이미지 변화 테스트
- 화면 라벨: `얼굴에 어울리는 스타일`
- 서버 값: `recommendationMode=face-fit`

## 2. 프리미엄 확장 상담

기본 헤어 추천은 무료 또는 기본 멤버십의 핵심 기능으로 유지한다. 코디와 메이크업은
결과의 상업적 가치를 높이는 확장 상담이므로 `premium_addons` 권한에 묶어 1회 추가
결제 또는 프리미엄 멤버십으로 제공한다.

### 코디 추천

- 타깃: 남성, 여성 모두
- 범위: 헤어스타일에 맞는 상의, 하의, 아우터, 액세서리, 컬러 무드, 상황별 스타일링
- MVP 반영: 헤어 추천 사유에 짧은 코디 방향을 포함
- 유료화: 상세 코디 카드, 계절별 착장, 쇼핑 링크, PDF 보고서로 확장
- 서버 값: `premiumAddOns=["outfit-coordination"]`
- 출력 필드: `outfitAdvice`
- 이미지 정책: 헤어 합성 단계에서는 의상을 강제로 바꾸지 않고, 별도 코디 이미지 생성 상품으로 분리한다.

### 메이크업 스타일 추천

- 타깃: 여성 우선
- 범위: 베이스 표현, 립 톤, 눈썹/아이 메이크업 밸런스, 전체 분위기
- MVP 반영: 여성 모드에서만 선택 가능하며, 헤어 추천 사유에 짧은 메이크업 방향을 포함
- 유료화: 얼굴형/퍼스널 컬러/헤어 컬러 기반 메이크업 리포트, 전후 비교 이미지, PDF 보고서로 확장
- 서버 값: `premiumAddOns=["makeup-style"]`
- 출력 필드: `makeupAdvice`
- 이미지 정책: 기본 상담에서는 메이크업 문장 조언만 제공하고, 얼굴 동일성 리스크가 낮아진 뒤 별도 전후 이미지 상품으로 확장한다.

## 2-1. 확장 서비스 기획안

### 상품 구조

- 기본 추천: 사진 2~3장, 성별 모드, 추천 목적, 헤어 컬러, 선호 컷을 반영해 헤어 9개 추천
- 프리미엄 스타일 리포트: 기본 추천에 코디 추천과 여성 메이크업 추천을 추가
- 멤버십 확장: 월별 상담 횟수, 히스토리 보관, PDF/export, 고해상도 저장, 코디/메이크업 상세 카드 제공
- 미용실용 확장: 고객별 상담 히스토리, 현재 기장 기준 상담 기본값, 디자이너 메모, 예약/리뷰/지도 연동

### UX 흐름

- 사진 업로드 후 `추천 목적`에서 현재 기장 기준 또는 얼굴 적합 기준 선택
- 원하는 헤어컷과 컬러를 선택하지 않으면 미리룩이 자동 추천
- 프리미엄 권한이 없으면 확장 상담 선택 버튼을 잠그고 PortOne 결제 CTA 노출
- 결제 또는 멤버십 권한이 있으면 코디/메이크업 항목을 추천 API payload에 포함
- 추천 카드, 상세 보기, 히스토리, 공유 보드, 이메일/PDF export에 확장 조언을 같은 필드로 노출

### 품질 기준

- 현재 기장 기준에서는 불가능한 장발/탈색/전환 스타일을 과도하게 추천하지 않는다.
- 얼굴 적합 기준에서는 현재 기장과 다를 수 있음을 솔직하게 표시한다.
- 코디와 메이크업은 헤어 선택을 보조하는 상담 조언이며, 얼굴 동일성과 헤어 합성 품질을 해치지 않는다.
- 여성 고객은 헤어, 컬러, 메이크업, 코디의 조화가 중요하므로 프리미엄 상품의 우선 타깃으로 둔다.
- 남성 고객도 코디 추천은 유효하므로 성별 공통 상품으로 둔다.

## 3. 결제 상품 구조

- 무료: 사진 업로드, 성별 선택, 헤어 추천 9개, 기본 스타일 미리보기
- 1회 추가 결제: 코디 추천 또는 메이크업 추천을 1회 상담에 추가
- 프리미엄 멤버십: 월간 상담 횟수, 고해상도 저장, PDF export, 코디/메이크업 리포트, 히스토리 보관 기간 확대
- 미용실용 멤버십: 현재 기장 기준 상담, 고객별 상담 히스토리, 상담 보드 저장, 예약/리뷰/지도 연동

## 4. 단계별 실행 계획

### Phase 1: 추천 프롬프트 반영

- 웹 UI에 추천 기준 2개를 노출
- API에 `recommendationMode`를 전달
- 프리미엄 확장 옵션을 추천 사유에 반영
- 히스토리에 선택 기준과 확장 상담명을 저장

진행: 추천 화면에서 `현재 기장으로 가능한 스타일`과 `얼굴에 어울리는 스타일`을
선택할 수 있게 했고, 서버 추천 프롬프트도 두 목적을 분리해 해석한다. 현재 기장
기준은 미용실 상담용, 얼굴 적합 기준은 개인 탐색용으로 안내한다.

### Phase 2: 구조화된 상담 카드

- 추천 API 응답에 `outfitAdvice`, `makeupAdvice`, `maintenanceAdvice` 필드 추가
- 추천 카드에서 헤어 사유와 확장 상담을 분리 표시
- PDF/export에 확장 상담 섹션 추가

진행: 추천 API schema에 `salonProcess`, `maintenanceAdvice`, `outfitAdvice`,
`makeupAdvice`를 추가했다. 웹 추천 카드, 선택 상세, 최근 히스토리, `/history`,
PDF 인쇄, 이메일 export, `/share/[token]` 공유 보드에서 같은 상담 조언을 표시한다.
서버 저장은 기존 `hairstyle_recommendations`의 `salon_process`, `caution` 컬럼을
활용해 production migration 없이 우선 호환되도록 했다.

### Phase 3: 결제/멤버십 연결

- PortOne 상품 ID를 `premium_addons`와 연결
- 결제 완료 시 해당 상담 세션에 확장 기능 활성화
- Supabase에 결제 상태, 사용 횟수, 상담 세션 권한 저장

진행: `premium-style-report` 상품을 추가했고, PortOne 결제 검증이 성공하면
`payment_events.profile_id`, `entitlement`, `entitlement_expires_at`에 계정 권한을
저장한다. `/api/payments/entitlements`는 로그인 사용자의 `premium_addons`,
`vote_boost`, `salon_pack` 활성 권한을 반환하며, 메인 추천 화면의 프리미엄 확장
패널이 이 상태를 표시한다. 메인 추천 화면에서는 `premium_addons` 권한이 없으면
코디/메이크업 확장 선택 버튼을 잠그고, 같은 패널에서 `premium-style-report`
결제 CTA를 노출한다. 결제가 기록되면 권한을 다시 조회해 화면 새로고침 없이 확장
상담을 열어 준다. `/api/hairstyles/recommend`도 `premiumAddOns`가 포함된 요청은
Supabase 계정의 활성 `premium_addons` 권한을 서버에서 다시 확인한 뒤에만 처리한다.

### Phase 4: 이미지 확장

- 코디/메이크업은 기본 헤어 합성 이미지에서 즉시 변경하지 않는다.
- 고객이 별도 “코디 이미지 생성” 또는 “메이크업 이미지 생성”을 선택할 때만 의상/메이크업 비주얼을 생성한다.
- 얼굴 보존과 헤어스타일 상담 품질을 우선하고, 확장 이미지는 별도 결과물로 분리한다.

## 5. 운영 원칙

- 미용실용 추천은 현재 기장 기준을 기본값으로 둔다.
- 개인 탐색용 추천은 얼굴 적합 기준을 선택하게 한다.
- 코디/메이크업 확장은 추천 품질을 보조하되, 헤어 합성 이미지의 얼굴 동일성과 상담 신뢰도를 해치지 않는다.
- 여성 고객은 헤어, 컬러, 메이크업, 코디가 함께 작동하므로 확장 상담을 별도 유료 상품으로 만들 가치가 높다.
